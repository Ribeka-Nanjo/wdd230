Hick's Low

Nike
https://www.nike.com/men

The design uses Hick's Low because the options in the menu are minimized to be easy to choose from for users. Also, it assists Users in manipulating and taking the following action smoothly.

Contrast

Apple
https://www.apple.com/watch/

First of all, we can see the size contrast in the texts. It helps make them readable and legible so that Users can quickly know what the page is they are looking for. Next, we can see the color contrast in the buttons. For example, the blue “buy” button is different from other action buttons. The design leads users to take action (buying or browsing.)

Alignment

Lush
https://www.lushusa.com

We can see beautiful alignment in the webs site. It repeats the same pattern of alignments and helps the design be clear and readable. Because grids neatly arrange the texts and products' pictures, they create unity in the design. It draws users' eyes and helps increase product desire.
